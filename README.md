## Мета роботи
Навчитися працювати з колекціями та реалізувати абстракцію доступу до даних у програмному застосунку з використанням патерну **Repository**.

## Завдання
1. Організувати зберігання об’єктів у колекціях.
2. Реалізувати інтерфейс `IRepository<T>`.
3. Реалізувати додавання, видалення та пошук об’єктів.
4. Відокремити логіку доступу до даних від інтерфейсу користувача.
5. Забезпечити коректну роботу застосунку

## Використані технології та підходи
- **Мова програмування:** C#  
- **Колекції:** `List<T>`, `Dictionary<TKey,TValue>`, `HashSet<T>`  
- **Generics:** Узагальнені інтерфейси та класи (`IRepository<T>`)  
- **Патерн Repository:** Для абстракції доступу до даних  
- **SOLID принципи:**  
  - Single Responsibility  
  - Open/Closed  
  - Liskov Substitution  
  - Interface Segregation  
  - Dependency Inversion  
- **Singleton:** Клас `GameManager`  
- **UI:** Windows Forms (`Form1`)
